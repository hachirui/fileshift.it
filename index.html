Come funziona

01
Carica il file
Trascina immagine, audio o video nell'area di caricamento. Nessun limite di formato.
02
FFmpeg si carica
La prima volta scarica FFmpeg.wasm (~30MB, poi in cache). Audio e video vengono elaborati localmente con la stessa potenza di FFmpeg desktop.
03
Converti e scarica
Premi Converti e attendi pochi secondi. Il file convertito si scarica direttamente nel tuo dispositivo.
04
Privacy totale
Tutto avviene nel browser via WebAssembly. Nessun file viene mai inviato a server esterni.

Domande frequenti

No. FileShift usa Canvas API per le immagini e FFmpeg.wasm per audio e video — tutto gira nel tuo browser tramite WebAssembly. I tuoi file non lasciano mai il tuo dispositivo.
Seleziona la categoria Video, carica il file MP4, scegli MP3 come formato di destinazione e clicca Converti. La prima volta FFmpeg.wasm si scarica (~30MB), poi rimane in cache e le conversioni successive sono immediate.
La prima volta viene scaricato FFmpeg compilato in WebAssembly (~30MB). Questo avviene una volta sola — le volte successive il browser lo carica dalla cache e la conversione parte quasi istantaneamente.
Non c'è un limite rigido, ma la conversione dipende dalla RAM disponibile nel tuo dispositivo. File fino a 500MB funzionano bene su computer desktop. Per file molto grandi su mobile, consigliamo di usare una connessione WiFi e un dispositivo con almeno 4GB di RAM.
